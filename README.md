### Settings (Place them on top of script)
```lua
getgenv().Compact = true; -- // Uses Mobile Variant for PC
```
```lua
getgenv().Multi_Support = true; -- // Multi Config Support (One Path)
```
```lua
getgenv().Autoload_Pause = true; -- // Stops autoload from running
```
