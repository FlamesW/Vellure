### Settings (Place them on top)

#### Uses Mobile Variant for PC
```lua
getgenv().Compact = true;
```
#### Multi Config Support (One Path)
```lua
getgenv().Multi_Support = true;
```
#### Stops Autoload from running
```lua
getgenv().Autoload_Pause = true;
```
#### Auto Update Library

** ⚠️ Cautious, Any update will reload the script
```lua
getgenv().Auto_Update = true;
```

#### Shutdown Library

```lua
getgenv().ShutDown();
```
